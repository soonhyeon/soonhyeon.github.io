Noisy-ArcMix

Deep Learning-Based Anomalous Sound Detection

📌 프로젝트 소개

Noisy-ArcMix는 Anomalous Sound Detection(ASD)을 위한 새로운 학습 기법으로, Additive Noisy Angular Margin Loss와 Mixup을 결합하여 일반적인 사운드 데이터를 더 강건하게 학습할 수 있도록 설계되었습니다.

본 프로젝트는 ICASSP 2024에서 발표되었으며, 코드와 데이터는 오픈소스로 공개되어 있습니다.

🚀 주요 특징

  • Mixup과 Angular Margin Loss를 활용한 이상 음향 탐지
  • 일반적인 환경에서도 강건한 사운드 이상 탐지 성능
  • ICASSP 2024 논문으로 발표된 최신 기술 적용

💻 설치 및 실행 방법


git clone https://github.com/soonhyeon/Noisy-ArcMix.git
cd Noisy-ArcMix
pip install -r requirements.txt
python train.py
